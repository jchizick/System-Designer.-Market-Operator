Signal degradation occurs when the system attempts to optimize for too many variables simultaneously.

At first, every new filter feels like protection. More confirmation. More context. More proof. Then the system begins to hesitate because too many inputs are competing for authority.

A clean signal has a hierarchy. Primary condition. Supporting context. Invalidation. Everything else is either useful structure or decorated noise.

The operator's job is to protect the signal from cleverness. Additions should reduce ambiguity, not create more places to hide from a decision.

If the process needs constant interpretation, the signal has already degraded.
